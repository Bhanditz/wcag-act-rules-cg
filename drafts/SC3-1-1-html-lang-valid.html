Rule: undefined

Please note, this is a draft

Late updated: May 14, 2018

Created by: Annika Nietzio, Jey Nandakumar

Description

This rule checks the lang or xml:lang attribute conforms to BCP 47.

Accessibility Requirements

This conformance rule relates to:

  • Success Criterion 3.1.1 (Language of Page)
    • How to Meet 3.1.1 (Language of Page)
    • Understanding Success Criterion 3.1.1 (Language of Page)

Test Procedure

Applicability

The root element of the page, if it is an html element with a lang and/or xml:lang attribute.

Expectation

The value of the lang or xml:lang attribute conforms to BCP 47.

Note: If lang and xml:lang exists, then only lang is tested for conformance.

Assumptions

There are currently no assumptions

Accessibility Support

There are known combinations of a popular operating system with browsers and assistive technologies that do not support the lang and xml:lang attributes.
