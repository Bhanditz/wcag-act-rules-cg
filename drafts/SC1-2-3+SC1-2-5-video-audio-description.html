Rule: audio described video element

Please note, this is a draft

Late updated: Jul 10, 2018

Created by:

  • Wilco Fiers
  • Brian Bors

Success_criterion:

  • 1.2.3 (Audio Description or Media Alternative (Prerecorded))
    • ,
  • 1.2.5 (Audio Description (Prerecorded))

Description

Non-streaming video elements must not contain any visual information that is not expressed in the audio

Accessibility Requirements

This conformance rule relates to:

  • Success Criterion 1.2.3 (Audio Description or Media Alternative (Prerecorded))
    • How to Meet 1.2.3 (Audio Description or Media Alternative (Prerecorded))
    • Understanding Success Criterion 1.2.3 (Audio Description or Media Alternative (Prerecorded))
  • Success Criterion 1.2.5 (Audio Description (Prerecorded))
    • How to Meet 1.2.5 (Audio Description (Prerecorded))
    • Understanding Success Criterion 1.2.5 (Audio Description (Prerecorded))

Test Procedure

Applicability

The rule applies to any non-streaming video element visible in the page where the video contains audio.

Expectation

The visual information of each test target is available through its audio, or through an audio description track.

Assumptions

There are currently no assumptions

Accessibility support

There are no major accessibility support issues known for this rule.
