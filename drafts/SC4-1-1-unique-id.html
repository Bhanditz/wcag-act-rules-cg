Rule: unique element id attributes

Please note, this is a draft

Late updated: Jun 1, 2018

Created by:

  • Bryn Anderson
  • Jey Nandakumar
  • Kamyar Rasta
  • Wilco Fiers

Success_criterion:

  • 4.1.1 (Parsing)

Description

This rule checks that all id attribute values on a single page are unique.

Accessibility Requirements

This conformance rule relates to:

  • Success Criterion 4.1.1 (Parsing)
    • How to Meet 4.1.1 (Parsing)
    • Understanding Success Criterion 4.1.1 (Parsing)

Test procedure

Applicability

Each test target has an id attribute that is unique within the document context of the element.

Expectation

All id attribute values are unique.

Assumptions

There are currently no assumptions.

Accessibility Support

There are no major accessibility support issues known for this rule.
