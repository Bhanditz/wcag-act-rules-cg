Rule: unique element id attributes

Late updated: Aug 6, 2018

Created by:

  • Bryn Anderson
  • Anne Thyme

Success_criterion:

  • 4.1.1 (Parsing)

Description

This rule checks that all id attribute values on a single page are unique.

Accessibility Requirements

This conformance rule relates to:

  • Success Criterion 4.1.1 (Parsing)
    • How to Meet 4.1.1 (Parsing)
    • Understanding Success Criterion 4.1.1 (Parsing)

Test procedure

Applicability

Any element that has an id attribute.

Expectation

Each test target has an id attribute value that is unique within the document context of the element.

Assumptions

There are currently no assumptions.

Accessibility Support

There are no major accessibility support issues known for this rule.
